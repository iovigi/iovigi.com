CSS Files and Structure - top

We are using sevaral CSS files in this template. We always follow the latest trends of coding standard. We make this template with the most popular CSS framework named Bootstrap. This tamplate is build with Bootstarp 3.

CSS File included:
  • bootstrap.min.css used
  • font-awesome.min.css used
  • style.css used
  • respnosive.css used

# bootstrap.min.css - Currently v3.3.5 is the main bootstrap css file.

JavaScript - top

This theme imports Following Javascript files.


  1. jQuery Library
  2. Bootstrap js scripts
  3. Instafeed js
  4. Smartmenu js
  5. Masonry JS Plugin used
  6. Google Map API and its js used

There are a few functions worth looking over.
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the customs scripts.
  3. In addition to the custom scripts named google map, we implement its plugin. This custom code used to control google map and navigation menu. For example:
